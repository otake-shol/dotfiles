# Claude Project Templates

プロジェクト種別ごとのCLAUDE.mdテンプレート集。

## 使い方

1. プロジェクトに合ったテンプレートをコピー
2. プロジェクトルートに `.claude/CLAUDE.md` として配置
3. プロジェクト固有の情報を追記

## テンプレート一覧

| ファイル | 用途 |
|----------|------|
| `nextjs-app.md` | Next.js アプリケーション |
| `cli-tool.md` | CLIツール開発 |
| `library.md` | npm/PyPIライブラリ |

※ 必要に応じてテンプレートを追加
