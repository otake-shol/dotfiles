# /seiri - ダウンロードフォルダ整理

iCloud Contents内の `01_ダウンロード` 配下のファイルを適切なフォルダに振り分けて空にします。

## 対象パス

- ベース: `/Users/otkshol/Library/Mobile Documents/com~apple~CloudDocs/Contents`
- 整理対象: `01_ダウンロード`

## 振り分けルール

| ファイルの種類 | 移動先 |
|--------------|--------|
| 資格・証明書（AWS、Oracle、PMP、CASEC等） | `10_自己紹介` |
| 結婚式関連（見積り、進行表、招待者リスト等） | `14_結婚式` |
| 税務・金融（源泉徴収、確定申告、贈与税等） | `12_事務` |
| 不動産・住宅（物件資料、住宅ローン、重説等） | `12_事務` |
| 連絡先（.vcfファイル） | `12_事務` |
| セキュリティ（1Password、リカバリフレーズ等） | `12_事務` |
| 学習・技術資料（書籍PDF、技術ドキュメント等） | `20_書籍・論文` |
| 写真・画像（.jpg、.png、.heic、動画等） | `11_写真` |
| キャリア・仕事（職務経歴書、キャリア関連等） | `15_キャリア` |
| 旅行（旅行プラン、Airbnb関連等） | `13_趣味` |
| 分類不能 | ユーザーに確認 |

## 手順

1. `01_ダウンロード` 配下のファイル・フォルダを一覧表示
2. 各ファイルを上記ルールに従って分類
3. 分類不能なファイルはユーザーに確認
4. ファイルを各フォルダに移動
5. 空になったサブフォルダを削除
6. 整理結果をサマリー表示

## 注意事項

- 移動前に必ずファイル一覧を確認
- 重複ファイル名がある場合はユーザーに確認
- .DS_Store等のシステムファイルは無視
- サブフォルダ構造がある場合はそのまま移動
