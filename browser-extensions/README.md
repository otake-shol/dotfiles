# ブラウザ拡張機能

ブラウザ別の拡張機能リストです。

## ファイル構成

- `arc.md` - Arc ブラウザ
- `chrome.md` - Google Chrome
- `firefox.md` - Firefox

## 使い方

### 拡張機能を確認

```bash
cat ~/dotfiles/browser-extensions/arc.md
```

### 新しい拡張機能を追加

1. ブラウザに拡張機能をインストール
2. 該当ファイルに追記:

```markdown
#### 拡張機能名
- **用途**: 何に使うか
- **インストール**: ストアURL
```

3. コミット
